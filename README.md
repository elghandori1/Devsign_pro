# Devsign Pro

Official portfolio and services website for Devsign.

## Live Website

https://devsignpro.com

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- i18n (English, French, Arabic)

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build for Production

```bash
npm run build
npm run start
```
